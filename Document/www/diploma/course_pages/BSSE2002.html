Tools in Data Science

This course will teach students to use popular tools for sourcing data, transforming it, building and optimizing models, communicating these as visual stories, and deploying them in production.

by S Anand

Course ID: BSSE2002

Course Credits: 2

Course Type: Data Science

Recommended Pre-requisites: None

Recommended Co-requisites: BSCS2004 -  Machine Learning Foundations

Course structure & Assessments

8 modules over 12 weeks of coursework, online assignments for each module, 1 remote online exam, 2 take home projects, 1 in-person end term exam. For details of standard course structure and assessments, visit Academics page.

MODULE 1 Discover the data
MODULE 2 Get the data
MODULE 3 Prepare the data
MODULE 4 Model the data
MODULE 5 Modern tools to simplify deep learning models
MODULE 6 Design your output
MODULE 7 Narrate a story
MODULE 8 Deploy the results
